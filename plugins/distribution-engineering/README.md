# Distribution Engineering (plugin)

The marketing-side analog of compound engineering — brand, distribution, and a
compounding content memory, as AI-agent skills.

See the [repository README](../../README.md) for the overview and install
instructions, and [`AGENTS.md`](AGENTS.md) for the operating philosophy, canonical
files, compounding loop, and skill roadmap.

## Skills

| Skill | Status | Does |
|-------|--------|------|
| `de-brand` | ✅ | Create/maintain `BRAND.md` — positioning, audience, voice, channels |
| `de-distribution` | planned | Channel playbook → `DISTRIBUTION.md` |
| `de-design` | planned | Visual identity → `DESIGN.md` |
| `de-ideate` / `de-write` / `de-content-review` | planned | The make loop |
| `de-compound` / `de-pulse` | planned | Log results, mine patterns — the compounding spine |

## Usage

```
/de-brand
```

Run it in any project to define (or update) that product's brand anchor. Every
other `de-*` skill reads `BRAND.md` as grounding.
